# Affinity Scripts

Automation scripts for Affinity using SDK

---

## Scripts

### Scale Up ×1.618 — Golden Ratio (φ)
`scale-up-phi.js`

Scales the selected object(s) up by φ (1.618...) from the centre of the selection's bounding box. Works on single objects or multi-object selections.

### Scale Down ×0.618 — Golden Ratio (1/φ)
`scale-down-phi.js`

Scales the selected object(s) down by 1/φ (0.618...) from the centre of the selection's bounding box. The inverse of the above — run either repeatedly to step through a proportional scale sequence.

### Centre Selection on Artboard
`centre-on-artboard.js`

Centres the selected object(s) — both horizontally and vertically — on the active artboard. On spreads with multiple artboards, the script targets whichever artboard contains the selection's centre point, falling back to the first artboard if none matches. Works with single or multi-object selections (centres the combined bounding box).

## How these scripts work

Affinity v3 scripting is AI-driven. Scripts are generated by Claude via the Affinity MCP connector and saved directly into Affinity's Scripts panel — there is no manual script editor or import function in the Affinity UI itself.

To use these scripts you have two options:

### Option A — Ask Claude to install them (recommended)

1. Set up the Affinity MCP connector in Claude: [affinity.studio/help/ai-connector-setup](https://www.affinity.studio/help/ai-connector-setup/)
2. Open a conversation with Claude and paste in the script code, asking Claude to save it to your Affinity script library
3. The script will appear in your Scripts panel (Window → General → Scripts) ready to run

### Option B — Use the community Affinity Script Manager

[Affinity Script Manager](https://github.com/JiriKrblich/Affinity-script-manager) is a third-party tool that provides a GUI for managing scripts outside of Claude. It supports drag-and-drop import of `.js` files directly into your Affinity library.

### Running a script

1. Open the Scripts panel: **Window → General → Scripts**
2. Select your object(s) in the document
3. Click the script name to run it

---

## Notes

- All scripts operate on the **current selection**. If nothing is selected, the script logs a message and exits cleanly.
- The scale scripts preserve aspect ratio — both axes are scaled by the same factor.
- Requires **Affinity by Canva v3.2.2** or later.

---

## Background

The golden ratio (φ ≈ 1.618) is a proportional system with a long history in design and typography. Scaling objects by φ or 1/φ produces a natural harmonic progression — the same ratio used in classical type scales, architectural proportions, and grid systems. These scripts make that progression available as a one-click operation, replicating the workflow previously set up in Illustrator via Transform → Scale with a custom keyboard shortcut.
